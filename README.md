# iPorty
Port based IP search python script

서버가 유동 IP라 가끔 IP가 바뀜..  
그래서 잘 사용되지 않은 포트로 ssh를 열어두고  
내가 사용하는 port를 기준으로 국내 IP대역 스캔을 돌림  
그러면 보통은 사용하지 않는 포트가 열려있는 곳에 ssh접속을하면 내 서버 IP를 찾을 수 있음
